1. tool dùng chromedriver, tải qua https://googlechromelabs.github.io/chrome-for-testing/
2. chạy file gui.py
3. account: điền acc của mình. format: username:password. nhiều acc thì mỗi acc 1 dòng
4. comment + like + save / report: để like + save + comment post / report. có thể chạy cùng lúc
5. tool chạy xong một tác vụ, sẽ không đóng trong 10p để có thể làm tác vụ khác mà không log out acc. sau đó sẽ hiện noti stop session. hoặc có thể đóng tool luôn
6. hdsd:

* comment box: lên chatgpt lấy một list comment (mỗi comment một dòng). nếu không muốn comment thì để trống để like post thôi
* account\_target1, account\_target2: tick để thao tác với acc của ling hoặc orm hoặc lingorm
* num of posts box: thao tác lên n post mới nhất của mỗi acc (>= 1). tức nếu điền 5 thì sẽ là 10 posts nếu tick cả 2
* media box: điền LINK post
* report box: format acp: instagram.com/... hoặc https://instagram.conm/... hoặc mỗi username cũng được
